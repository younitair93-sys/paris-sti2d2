# 🎰 Paris de Classe - 1STI2D2

Application de paris collaborative pour la classe 1STI2D2

## 🚀 Démarrage rapide

### Installation
```bash
npm install
```

### Développement
```bash
npm run dev
```

### Build
```bash
npm run build
```

## 📝 Fonctionnalités

### Joueurs
- ✅ Connexion avec pseudo
- ✅ Solde initial de 500 💰
- ✅ Placement de paris sur les questions ouvertes
- ✅ Visualisation du classement
- ✅ Historique des paris terminés

### Admin
- ✅ Créer des paris avec 2 options
- ✅ Fermer les paris et désigner le gagnant
- ✅ Voir le classement en temps réel
- ✅ Supprimer les paris (admin complet)

## 🎮 Comment ça marche

1. Les joueurs se connectent avec un pseudo
2. Ils reçoivent 500 💰 de solde initial
3. Les admins créent des paris avec 2 options
4. Les joueurs misent sur une option
5. L'admin ferme le pari et désigne le gagnant
6. Les gains sont distribués proportionnellement

## 💾 Stockage

Les données sont sauvegardées localement via `window.storage` (Codesandbox compatible)

## 📱 Responsive

L'app est optimisée pour mobile et desktop
